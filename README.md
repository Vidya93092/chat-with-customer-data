# Chat with Customer Data

## Overview
AI-powered data analyst tool using Gemini AI and Pandas.

Users can upload Excel files and ask natural language questions.

## Features
- Upload Excel file
- Ask questions in English
- AI-powered answers
- Data analysis using Pandas
- Streamlit UI

## Tech Stack
- Python
- Streamlit
- Pandas
- Gemini API

## Run Project

pip install -r requirements.txt

streamlit run app.py

## Example Questions
- Total customers
- Average budget
- Customers in Pune
- Budget above 50 lakhs
